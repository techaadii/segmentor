# Segmentor

A human-in-the-loop in-context learning framework for semantic segmentation.

## Installation

### Using `pip`

Install the package from GitHub with `pip`

```bash
pip install "git+https://github.com/sattwik-sahu/segmentor.git"
```

### Using `uv`

Install the package from GitHub with `uv`

```bash
# with `uv add`
uv add "segmentor @ git+https://github.com/sattwik-sahu/segmentor.git"

# ... or with `uv pip`
uv pip install "git+https://github.com/sattwik-sahu/segmentor.git"
```
